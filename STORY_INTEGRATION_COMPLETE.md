## ✅ 已完成的任务

- ✅ 创建 `react-ts-demo/data-by-origin/scripts/` 用于存放数据处理脚本

### 2. 数据转换脚本
- ✅ 创建 `convertStoriesToJSON.js` 脚本
- ✅ 实现 CSV 到 JSON 的自动转换
- ✅ 处理三种数据源：
  - `section-stories/` - 分段故事文本
  - `sentence-stories/` - 句子级故事文本
  - `questions/` - 理解问题

### 3. 难度分类算法
实现了智能难度分类系统，基于以下因素：
- **故事长度** (30% 权重)
  - < 500 词：10分
  - 500-900 词：25分
  - 900-1300 词：35分
  - > 1300 词：45分

- **问题复杂度** (40% 权重)
  - 隐式问题比例 < 45%：15分
  - 隐式问题比例 45-65%：30分
  - 隐式问题比例 > 65%：45分

- **问题数量** (30% 权重)
  - < 18 题：10分
  - 18-28 题：25分
  - > 28 题：35分

- **集合调整**
  - 彼得兔系列、基础故事集：-20分
  - 安徒生童话、格林童话等：-10分

**最终难度分布：**
- 简单：36 个故事 (13%)
- 中等：96 个故事 (35%)
- 困难：146 个故事 (52%)

- ✅ `all-stories.json` - 完整故事数据（包含文本、问题、统计信息）
- ✅ `story-index.json` - 故事索引（用于列表展示，减小加载体积）

## 🚀 使用方法

### 1. 访问故事列表
- 从少儿英语首页点击"绘本故事"
- 或直接访问 `/stories` 或 `/child-stories`

### 2. 筛选故事
- 点击难度筛选按钮（全部/简单/中等/困难）
- 查看每个难度级别的故事数量

### 3. 阅读故事
- 点击故事卡片查看详情
- 点击"开始阅读"进入阅读器
- 使用导航按钮切换段落
- 点击单词查看详情和发音
- 点击"朗读"按钮听故事
- 点击"X道题"查看理解问题

### 4. 重新生成数据（如需更新）
```bash
cd react-ts-demo/data-by-origin/scripts
node convertStoriesToJSON.js
```
